## Requerimientos

## Utilizacion

* Clonar el repositorio

* Ir a la carpeta del proyecto y abrir la consola

* Instalar las dependencias

```text
npm install
```

* Correr el servidor

```text
npm run start
```

* Ir a la pagina en `http://localhost:3000`
